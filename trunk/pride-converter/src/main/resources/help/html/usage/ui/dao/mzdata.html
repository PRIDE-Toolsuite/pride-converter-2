mzData DAO Options

Previous | Next | See Also

mzData is the PSI's previous standard file format for MS spectra data. A detailed documentation about the mzData file format can be found at the PSI's homepage (http://www.psidev.info/index.php?q=node/80#mzdata).

Details

PRIDE XML files store spectra data using the mzData format internally. Thus, the data from mzData files is directly copied to the PRIDE XML file. mzData files do not contain identifications.
