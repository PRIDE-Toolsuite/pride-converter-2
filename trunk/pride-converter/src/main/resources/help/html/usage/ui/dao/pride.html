PRIDE DAO Options

Previous | Next | See Also

The PRIDE DAO is based on the PRIDE JaxB library. It is used by the PRIDE XML Merger and the PRIDE XML Filter tools.

Details

The PRIDE DAO does not support any configuration options.


Optional Settings

These settings may be present or not, depending on which tool is used.

Compress Output File Specifies if the output file generated by the tool should be compressed using GZIP.
