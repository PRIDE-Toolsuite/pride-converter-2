File Selection

Previous | Next | See Also

This screen allows you to assign the proper mzTab file to the source file that corresponds to it.

In the case that you have multiple source files and multiple mzTab files as part of a single conversion, this form will enable you to map the correct mzTab file to the appropriate source file. If you have used the PRIDE mzTab generator to generate your mzTab files, it will have used the default file naming convention of [SOURCE_FILE]-mztab.txt. PRIDE Converter will try and automatically pair up source files and mzTab files based on the default file naming convention.

If the mzTab files do not follow the default file naming convention, you will need to choose from the pull-down menu and assign the proper file manually.
