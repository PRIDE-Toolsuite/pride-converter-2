Format Selection

Previous | Next | See Also

This is the first screen for PRIDE Converter. In this screen, you can select the file format you wish to convert into PRIDE XML. A short description of the format will be displayed when you select a file format.

Input format selection is the first choice that the user must make. When selecting a format, the user will be given information on the exact file type(s) required, as well as links to the vendor and/or format specification documentation. Please note that Mascot conversion is not available on the Mac OS platform, due to limitations imposed by the Mascot API itself. Although PRIDE Converter allows you to convert several files at once, they must be in the same file format.

PRIDE Converter currently supports the following formats:
