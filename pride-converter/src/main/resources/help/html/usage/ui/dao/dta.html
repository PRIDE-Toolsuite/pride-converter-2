The DTA DAO is based on the dta-parser by Griss et al. which is part of the jmzReader library (http://jmzreader.googlecode.com). A description about the dta file format can be found at the Matrix Science Help page. (http://www.matrixscience.com/help/data_file_help.html).

Details

The DTA DAO does not support any configuration options. DTA files only contain spectra information and no identification data.

The DTA DAO can process two types of inputs: either concatenated DTA files where spectra a separated by an empty line or directories that contain DTA files with one single spectrum each. Directories of concatenated DTA files cannot be added at once but the various DTA files must be added separately.
