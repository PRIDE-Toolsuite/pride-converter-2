PRIDE XML Merger Master File Selection

Previous| Next | See Also

This form allows you to select the master file during the XML merging process

A master file must be selected. All metadata annotations will be taken from this file to as a basis to generate the merged PRIDE XML file. The only file-specific information that will be retained from all other XML files will only be the identifications and spectra.
