Instrument Description

Previous | Next | See Also

This screen allows you to provide as much information as possible to describe your instrumentation setup.

The instrument description form is another form that can use templates, as in most cases, the instrumentation available to a user will not change dramatically over time. “Source”, “Analyzer” and “Detector” elements need to be annotated, preferably with CV terms from the Proteomics Standards Initiative (PSI) Mass Spectrometry (MS) controlled vocabulary (http://psidev.cvs.sourceforge.net/viewvc/psidev/psi/psi-ms/mzML/controlledVocabulary/psi-ms.obo).

The instrument name is mandatory, and users must provide at least one of each "Source", "Analyzer" and "Detector"
