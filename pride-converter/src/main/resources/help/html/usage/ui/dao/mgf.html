MGF DAO Options

Previous | Next | See Also

The MGF DAO is based on the mgf-parser by Griss et al. which is part of the jmzReader library (http://jmzreader.googlecode.com). A description of the mgf file format can be found at the Matrix Science Help page (http://www.matrixscience.com/help/data_file_help.html).

Details

The MGF DAO does not support any configuration options. MGF files only contain spectra information and no identification data.
