MS2 DAO Options

Previous | Next | See Also

The MS2 DAO is based on the ms2-parser by Griss et al. which is part of the jmzReader library (http://jmzreader.googlecode.com). A description of the ms2 file format can be found at the crux project page (http://noble.gs.washington.edu/proj/crux/ms2-format.html).

Details

The MS2 DAO does not support any configuration options. MS2 files only contain spectra information and no identification data.
