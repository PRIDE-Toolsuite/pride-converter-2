Format Selection

Previous | Next | See Also

This is the first screen for PRIDE Converter. In this screen, you can select the file format you wish to convert into PRIDE XML. A short description of the format will be displayed when you select a file format.

Although PRIDE Converter allows you to convert several files at once, they must be in the same file format.

PRIDE Converter currently supports the following formats:

  • mzML
  • mzIdentML
  • mzXML
  • Mascot
  • X|Tandem
  • mzData
  • Peaklist formats (MGF, MS2, DTA, PKL)

Note that some peaklist formats come in multiple versions, depending on whether the source files contain a single spectrum per file or multiple spectra per file.
