PRIDE Converter

Previous | Next | See Also

PRIDE Converter is an open source application to convert vendor-specific formats or PSI community standards into XML files suitable for submission to PRIDE.

Key Features:
  • Possibility to converty multiple files at once.
  • Low memory footprint, even for arbitrarily large source files.
  • Scriptable and interactive use are both possible.
  • A host of programmable open source API/libraries.
